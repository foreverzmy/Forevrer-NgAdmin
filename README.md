# Forevrer-NgAdmin

基于 Angular4+ 、 Angular Material 和 Bootstrap 的后台管理程序。

## 项目依赖

```
$ npm i -g @angular/cli
```

## 项目运行

```
$ npm start
```

## 功能列表

- [x] 登录/注销
- [x] Header 头部展示
- [x] 自动化可伸缩侧边栏
- [x] 自动化面包屑导航
- [ ] 路由守卫
- [x] 模板驱动表单
- [x] 响应式表单
- [x] 列表页
- [ ] 图片处理和上传
- [ ] 图表
- [x] Tabs
- [x] Dialog
- [x] Notification
- [ ] 富文本编辑器
- [ ] Theme
